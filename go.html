About Go

Go (Golang) is a typed, compiled language made by Google. It’s simple, fast, and good for running many tasks at once.

Go’s easy syntax and quick compile make it good for big, fast programs in cloud and network systems.

  • simple syntax
  • supports running many tasks at once
  • fast to compile and run
  • has useful standard tools
